HTML

  • Head element is a container for metadata. Not visible to users
  • Body contains content shown to users. Anything users see and use must be in here.
  • DOCTYPE tells browser what document type we're using. Ex. HTML
  • Meta element contains info used by the browser
  • For more, see notebook

CSS

  • CSS- Cascading Style Sheets
  • Margin indicates space aroung the outside of an element
  • Padding indicates space around inside of an alement
  • * is a special selector that declares ALL elements will have that same style

Git

  • Git commands always start with git.
  • git checkout -b branch-name: makes a new branch and switches you to it
  • git status tells you what branch you're currently in
  • ls: list of files and directories
  • cd: change directories
  • cd ..: move backwards 1 directory
  • mkdir: create directory
  • for more, see notebook
